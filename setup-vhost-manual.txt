2. Configuration → Virtual Hosts → Add

   Domains: centraldispatch.id, www.centraldispatch.id
   Document Root: /root/centraldispatch
   Save

3. Virtual Hosts → CentralDispatch → Script Handler → Add
   Suffixes: *

4. Configuration → Listeners → Default → View → Edit
   В списке Virtual Host Mappings:
   - Убедитесь что CentralDispatch добавлен для centraldispatch.id, www.centraldispatch.id
   - Или добавьте вручную:
     Virtual Host: CentralDispatch
     Domains: centraldispatch.id, www.centraldispatch.id
